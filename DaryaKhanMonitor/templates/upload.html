File Format Requirements

Your file must contain the following columns (case-sensitive):

  • name - Student's full name
  • roll_number - Student's roll number (must be unique)
  • class - Student's class (e.g., 9, 10, 11, 12)
  • section - Student's section (e.g., A, B, C)
  • parent_name - Parent or guardian's name
  • parent_contact - Parent's phone number
Example CSV Format:
name,roll_number,class,section,parent_name,parent_contact
Ali Ahmed,101,9,A,Ahmed Khan,03001234567
Sara Bibi,102,9,A,Muhammad Ali,03009876543
Hassan Ali,201,10,B,Ali Hassan,03451234567
Note: If a student with the same roll number already exists, their information will be updated.
